Anatomía de una URL
Una URL (Uniform Resource Locator) es la dirección única de cada recurso disponible en la Web: una página, una imagen, un archivo de audio o cualquier otro tipo de contenido. Conocer su estructura es fundamental para trabajar con proyectos web.
Las tres tecnologías de la Web
Toda página web está construida con tres tecnologías que trabajan juntas de forma coordinada. Cada una tiene un rol preciso e irremplazable. Entender esta separación de responsabilidades es clave para desarrollar sitios web bien organizados.
Frontend vs. Backend
En el desarrollo web profesional existen dos grandes áreas de trabajo. Entender la diferencia nos ayuda a ubicarnos dentro del campo y a comprender qué parte del sistema estamos construyendo.
